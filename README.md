# Typewriter
A typing game made with pygame

Easy mode selects from 1000 common English words, while Hard mode selects from 70,000+ English words.
Type words as they go across the screen, before they reach the end of the screen. Words will be highlighted as you type them.
Your remaining allowed missed words will be displayed by stars in the upper right corner; miss 5 words and it's game over. 

Press 1 at anytime during the game to pause the game. Press 1 again to resume the game.
